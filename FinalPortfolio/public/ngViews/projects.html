Lyrical

|

Github

Node.js | Angular | Sinatra | Ruby | Materialize | HTML5 | CSS3

Search for lyrics, save them to a playlist and contribute to our open source database by adding lyrics of your choice. Lyrical provides a Spotify experience for users hoping to find and save lyrics. This app was built using Sinatra and Ruby for the server side and Node.js and Angular for the client side.



RSVP

|

Github

Node.js | Javascript | JQuery | Skeleton | HTML5 | CSS3

Browse through user created events, create your own events in our open source database, and update your attendance status so you and others can see who is coming to the event! RSVP provides a mechanism for users to easy plan events and track attendance. This app was built using Node.js, Javascript and JQuery. It was stylized using Skeleton, HTML5 and CSS3.









Urban Garden Fall

|

Github

Javascript | JQuery | HTML5 | CSS3

Catch as many flowers as you can to advance through all the levels. To catch a flower simply click or tap the flowers. As you advance through the 5 levels, flowers fall faster but are worth more points. This game was built using Javascript, JQuery, HTML5, and CSS3.



Connect

WordPress | WordPress Engine | WP-glogin | HTML 5

At Green Dot Public Schools (a charter management organization) employees constantly share information that changes from year to year. Green Dot's Knowledge Sharing Platform was inflexible, hard to navigate, and required a lot maintenance from the IT and Application teams. As the Technical Project Manager, I was tasked to vet applications that would meet our needs and fill these existing gaps. After 12 months of research, design, and build, my team and I launched a brand new Knowledge Sharing Platform on WordPress that incorporated a sleek design, was mobile friendly, improved processes, and received universal buy in from our national organization.
